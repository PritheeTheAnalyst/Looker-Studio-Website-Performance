## Project Description 
This project is an interactive sales analytics dashboard built using Google Looker Studio to analyze a brands's website performance. It is designed to analyze user traffic, engagement, and performance across different traffic sources and devices. This analysis checked bounce rate, traffic source and user engagement on the website.

## 🎯 Objectives
1. Analyze traffic sources and user behavior
2. Identify high-performing and low-performing channels
3. Track bounce rates and engagement levels
4. Provide insights to improve conversion and retention

## 📁 Dataset
The dataset includes website analytics data such as:
- Traffic sources (Organic, Paid, Referral, Social, Direct)
- Session duration
- Conversion rates
- Bounce rates
- Page views

## Tools Used
- Google Looker/ Data Studio
- Google Sheets

## 💡 Recommendations
- Optimize landing pages for paid traffic to reduce bounce rate

- Improve targeting strategies for paid campaigns
- Focus on high-performing channels for better ROI
